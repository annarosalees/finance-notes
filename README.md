# finance-notes

FX・CFD・バイナリーオプションの仕組みを、実務者の視点で整理した入門ノート。
各商品が「何か」「なぜその価格・その動きになるのか」を、専門用語に頼りすぎず
説明することを目指しています。

## 対象

- これから金融（FX/CFD/デリバティブ）の実務に関わる人
- 各商品の基礎とオペレーションの実際を、体系的に知りたい人

## 収録ノート

- `notes/cfd.md` … CFD（差金決済取引）：指数・商品・米国株・ETF など
- `notes/fx.md` … FX（外国為替証拠金取引）※作成予定
- `notes/binary.md` … バイナリーオプション ※作成予定

## 各ノートの構成

すべてのノートは共通の型で書いています。

1. その商品は何か
2. どのような仕組みで価格・損益が決まるか
3. 実務上どのように扱われるか（レート生成・カバー等の観点）
4. つまずきやすいポイント

## 注記

- 本ノートは一般的な金融知識の解説を目的とした個人的な学習記録であり、
  特定の企業・システムの内部情報を含みません。
- 投資助言を目的とするものではありません。
